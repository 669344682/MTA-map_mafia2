с первой папкой нужно сделать так, как она называется, переименовать на test и кинуть на диск C

CollEditor2 - это программа для получения коллизии

default.txt - список текстур

DeleteZeroAreaFaces.ms - удаляет невидимые стены у коллизии объекта, переместить в скрипты 3d max и запустить на выделенном объекте

FrameRes_Editor - это программа для получения положение объектов, нужно открывать sds файлы, и копировать часть которая начинается с Object,
потом надо вставить то, что скопировал в файл frame.txt, там при помощи notepad++ заменить все запятые на точки и символ " на пустоту, потом выделить (Name: )
с пробелом и заменить на obj|, так нужно чтобы с помощью проги SciTE которая устанавливается в LuaForWindows_v5.1.5-52, в файл записалось название объекта
и его положение, которое будет в файле frame_new.txt

josbalcaen_batch_exporter_importer - это скрипт для массового импорта объектов в 3ds, гайд как установить в интернете

KAM's_GTA_Scripts_Upd_by_GF_v0.3.7 - скрипты для экспорта объектов и коллизии

lua_mtl_obj - код для чтения файлов с расширение .obj, которые должны быть на диске С, в файле с расширение .mtl вместо hash текстуры записывается её реальное 
название, в программе SciTE откройте его и запустите нажав run program

LuaForWindows_v5.1.5-52 - поставьте галочку на установку SciTE, установится программа при помощи которой можно запускать lua скрипты

M2Toolkit_v2.001122019 - много возможностей, я в ней смотрю какие объекты мне экспортировать, если FrameRes_Editor не смог открыть файл, копирую положение вручную,
там справа есть надпись Matrix, нужно открыть её и там будет строка position, копируйте её, также не забудьте про ротацию, а потом замените запятые на точки и поставьте
вместо y: z: тоже запятые, x: удалите, а также в настройках в библиотеке указать 3 файла с расширеним .mtl, которые находятся в MafiaII\edit\materials

Model_Viewer - для экспорта объектов, экспортируйте на диск С, так как этот путь указан в скрипте lua_mtl_obj 

pos_write - запускайте скрипт через SciTE, он запишит имя и положение объекта в файл frame_new.txt

tt - запускайте скрипт через SciTE, он найдет похожие объекты и закомментирует их записав в файл frame_new_copy.txt, потом это надо будет вставить в таблицу object
в скрипте lua_mtl_obj, чтобы не было похожих объектов
